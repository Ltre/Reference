public static boolean $need true
#

true-需要, false-不需要 运行机制：执行某个XXDo::yyFunction()时，会在本类的两个Map【getNeedsMap和getWithoutMap】中扫描“XX/yy”已配置的过滤器需求。 从中找出需要和不需要的过滤器，如需要{A,B,C}，不需要{B,C,D}， 而后根据self::$need的值决定配置冲突的过滤器最终是否需要，如本例中冲突的有{B,C},则剔除之。 最终结果是{A}，算法与集合的减运算相同。
